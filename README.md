# Mellow — Request Creation (Desktop Screen)

## Overview

React (Vite) implementation of the **Request Creation – Edit** screen based on Figma, with a small **design system** layer (tokens + primitives).

## Features

The application provides a comprehensive interface for creating and editing requests with the following capabilities:

- **Request Parameters Configuration**: Set up role requirements, experience levels, skills, and languages
- **Budget Management**: Configure budget ranges with negotiable options
- **Timeline Settings**: Define project timelines with flexibility options
- **Description Generation**: AI-powered description generation with manual editing capabilities
- **Preview Mode**: Review the complete request before submission

## Design System

The project includes a modular design system with reusable components:

- **Primitives**: Button, Chip, Input, Modal, RadioCard, Toggle
- **Typography**: Consistent text styling across the application
- **Tokens**: Design tokens for colors, spacing, and typography

## Run locally

```bash
npm install
npm run dev
```
