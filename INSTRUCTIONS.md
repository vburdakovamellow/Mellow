# Инструкция по запуску приложения из ветки create-request-proto

## Шаги для запуска:

### 1. Переключиться на ветку
```bash
git fetch origin
git checkout create-request-proto
```

### 2. Установить зависимости (если еще не установлены)
```bash
npm install
```

### 3. Запустить приложение в режиме разработки
```bash
npm run dev
```

### 4. Открыть в браузере
Приложение будет доступно по адресу, который покажет Vite (обычно `http://localhost:5173`)

---

## Альтернативный вариант (если репозиторий еще не клонирован):

```bash
# Клонировать репозиторий
git clone <repository-url>
cd Mellow

# Переключиться на нужную ветку
git checkout create-request-proto

# Установить зависимости
npm install

# Запустить приложение
npm run dev
```

---

## Полезные команды:

- **Остановить сервер**: `Ctrl + C` в терминале
- **Собрать для продакшена**: `npm run build`
- **Просмотреть продакшен билд**: `npm run preview`
